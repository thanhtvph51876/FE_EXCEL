# Privacy Policy

ExcelAI uses uploaded files only to perform the tasks requested by the user, such as cleaning data, reconciliation, reports, formulas, VBA, and document generation.

We do not sell user data. We do not make user files public.

AI providers such as Gemini may be used to process user requests. Uploaded file content should be treated as sensitive business data by users before submission.

Users can delete their uploaded files from the application. Files and generated outputs may be retained according to the configured retention policy for operational recovery and audit needs.

Passwords, JWT tokens, API keys, and secrets are not intentionally logged.
